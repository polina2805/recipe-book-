# Recipe Book - Сайт кулинарных рецептов

Как установить и запустить этот сайт у себя на компьютере

## Шаг 1: Скачать проект
1. Откройте сайт GitHub: https://github.com/polina2805/recipe-book
2. Нажмите зеленую кнопку 'Code'
3. Выберите 'Download ZIP'
4. Распакуйте архив в папку на компьютере

## Шаг 2: Открыть проект в PyCharm
1. Запустите PyCharm
2. Файл → Открыть → Выберите папку 'recipe-book'
3. Дождитесь открытия проекта

## Шаг 3: Открыть терминал в PyCharm
Внизу окна PyCharm нажмите на вкладку 'Terminal'

## Шаг 4: Создать виртуальное окружение
В терминале напишите команду и нажмите Enter:

python -m venv venv

## Шаг 5: Для Mac/Linux напишите команду:
source venv/bin/activate

## Шаг 6: Установить библиотеки
В терминале напишите команду:

pip install Django==4.2.0 Pillow==10.0.0

## Шаг 7: Запустить миграции базы данных
В терминале напишите команду:

python manage.py migrate

## Шаг 8: Запустить сайт
В терминале напишите команду:

python manage.py runserver

## Шаг 9: Открыть сайт в браузере
Откройте любой браузер (Chrome, Firefox)

В адресной строке напишите: http://127.0.0.1:8000

Нажмите Enter

## Требования к системе:
- Python 3.8 или выше
- 500 МБ свободного места
- 1 ГБ оперативной памяти
- Современный браузер (Chrome 90+, Firefox 88+, Safari 14+)
